## The Core Insight

Claude doesn't automatically add "the good stuff" - it builds exactly what you ask for. If you ask for a component, you get a component. If you ask for a production-ready component with error handling and edge cases, you get that instead.

**The difference is in the prompt, not in Claude's capabilities.**
