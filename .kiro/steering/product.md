# Product Overview

## Child Support Calculator - B2B Lead Generation Platform

**Core Product:** Free Australian child support calculator that helps parents calculate payments according to the official Services Australia 8-step formula.

**Business Model:** B2B lead generation platform connecting parents with family law practitioners. Parents use the free calculator, complexity detection identifies high-value cases, and lawyers pay per qualified lead ($50-100/lead target).

**Target Users:**
- **Primary:** Australian parents needing child support calculations
- **Secondary:** Family law firms seeking qualified leads

**Key Features:**
- Real-time child support calculations using official government formula
- Complexity detection system (high variance, court dates, zero payments, multiple children)
- Change of Assessment (CoA) reason identification for special circumstances
- Lead capture forms with pre-populated calculation data
- Admin panel for lawyers to manage leads

**Platform Support:** 
- Native iOS/Android via Expo
- Web version deployed at auschildsupport.com
- Cross-platform React Native codebase

**Current Phase:** Phase 3A - Proof Before Pitch (Validation)
**Success Metric:** >2% click-through rate on "Get Legal Help" buttons
**Revenue Target:** $10K-$15K/month at scale (Year 1)

## User Journey
1. Parent enters calculation details (incomes, children, care arrangements)
2. Live calculation provides instant results
3. Complexity detection runs automatically
4. Smart conversion footer displays priority-based call-to-action
5. Inquiry form captures lead with full context for lawyer handover
